# Changelog

The authoritative, per-release changelog for this project lives on GitHub Releases:

**https://github.com/DitchOoM/buffer/releases**

Each published version (tags `vX.Y.Z`) has release notes generated from the merged
pull requests for that version. This project adheres to
[Semantic Versioning](https://semver.org/); PR labels (`major`, `minor`, or patch)
drive the version bump at merge time.

To see what changed between two versions locally:

```bash
git log --oneline v6.13.1..v6.14.0
```

> This file previously duplicated release notes by hand and drifted out of date.
> GitHub Releases is now the single source of truth.
